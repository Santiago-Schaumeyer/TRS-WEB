# Taller Metalúrgico TRS - Sitio Web 

Proyecto web desarrollado para el **Taller Metalúrgico TRS**, una empresa familiar con más de 25 años de trayectoria en el sector industrial. El objetivo principal es ofrecer una presencia digital profesional, permitiendo a los clientes potenciales solicitar presupuestos y conocer los servicios de mecanizado y tornería.

## 🚀 Características
- **Diseño Responsive:** Adaptado para verse perfecto en celulares, tablets y escritorio.
- **Formulario de Contacto:** Integración profesional con *Formspree* para recepción de presupuestos.
- **Galería interactiva:** Visualización dinámica de trabajos realizados.
- **Enlaces rápidos:** Acceso directo a contacto vía WhatsApp y Email.

## 🛠 Tecnologías utilizadas
- **HTML5:** Estructura semántica.
- **CSS3:** Estilos personalizados, Flexbox y diseño responsivo.
- **JavaScript:** Lógica para el manejo de la galería (Lightbox) y navegación.
- **Formspree:** Gestión de formularios de contacto.
